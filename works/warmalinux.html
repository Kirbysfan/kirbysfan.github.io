Warma Linux

A fan made Warma (A very cute and famous Bilibili user) themed system.
Ver.1 was based on Xubuntu 16.04, while Ver.2 was based on Linux Mint 18.3
Since this is mainly for China so you'll see a lot of Chinese software.

In the January of 2020, I was staying at home because of the epidemic,
and just at that time, I discovered Warma. Because of her cuteness, she
sooner became my favourite person. I also entered a QQ group full of Warma fans.
A few days later, I saw someone in the group made a Warma themed system based on Windows 10,
this aroused my curiosity, I started wondering if I can make a Warma themed Linux...
And because of that, I started making Warma Linux, with almost no experience distributing Linux.
This is my very first work that was related to Computer, which made me become more and more interested
to programming and many other IT related things.

Known Bugs

Ver.1

  1. Uncustomized default theme.
  2. Only contains Simplified Chinese language, so you'll have to set language again after installing. (If you're not going to use that)
  3. MBR users will have to use boot-repair to fix boot after installing.

Ver.2

  1. Same as above.
  2. Some programs might fail to start or become corrupted.
